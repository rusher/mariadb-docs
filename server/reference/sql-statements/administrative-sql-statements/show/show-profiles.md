# SHOW PROFILES

## Syntax

```sql
SHOW PROFILES
```

## Description

The `SHOW PROFILES` statement displays profiling information that indicates resource usage for statements executed during the course of the current session. It is used together with [SHOW PROFILE](show-profile.md).
