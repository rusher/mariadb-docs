# WSREP\_LAST\_WRITTEN\_GTID

## Syntax

```sql
WSREP_LAST_WRITTEN_GTID()
```

## Description

Returns the [Global Transaction ID](../../../../ha-and-performance/standard-replication/gtid.md) of the most recent write transaction performed by the client.
