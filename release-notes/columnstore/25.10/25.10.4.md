# MariaDB ColumnStore 25.10.4 Release Notes

**Release date:** 17 Mar 2026

MariaDB ColumnStore 25.10.4 is a [Stable (GA)](../../community-server/about/release-criteria.md) release of MariaDB ColumnStore.

## Issues fixed

### Can result in hang or crash

* \[[MCOL-5987](https://jira.mariadb.org/browse/MCOL-5987)] - DROP TABLE doesn't release OID bit in oidbitmap
* \[[MCOL-4270](https://jira.mariadb.org/browse/MCOL-4270)] - DROP COLUMN IF EXISTS fails if the column is already present
* \[[MCOL-6083](https://jira.mariadb.org/browse/MCOL-6083)] - Post upgrade to 23.10.4, some queries are taking forever to execute..’
* \[[MCOL-6088](https://jira.mariadb.org/browse/MCOL-6088)] - group by alias improvement: ERROR 1178 (42000): The storage engine for the table doesn't support MCS-2016: Non supported item on the GROUP BY list.
* \[[MCOL-6243](https://jira.mariadb.org/browse/MCOL-6243)] - cpimport fails at fields longer than 1M like longtext and MariaDB ColumnStore crashes with "Broken pipe"
* \[[MCOL-6093](https://jira.mariadb.org/browse/MCOL-6093)] - Multiple PUT /config operations had arrived from primary CMAPI node shutting down a cluster

### Related to performance

* \[[MCOL-5983](https://jira.mariadb.org/browse/MCOL-5983)] - Bit-wise operations slower than expected

### Notable changes

* \[[MCOL-6151](https://jira.mariadb.org/browse/MCOL-6151)] - Query Accelerator select filter support
* \[[MCOL-6156](https://jira.mariadb.org/browse/MCOL-6156)] - Query Accelerator TPCH Q19 fix
* \[[MCOL-6157](https://jira.mariadb.org/browse/MCOL-6157)] - Query Accelerator TPCH Q7 fix
* \[[MCOL-6251](https://jira.mariadb.org/browse/MCOL-6251)] - Query Accelerator support subqueries
* \[[MCOL-6268](https://jira.mariadb.org/browse/MCOL-6268)] -  Query Accelerator TPC-H queries with subslect with aggregate are not working (queries 2, 11 and 18)
* \[[MCOL-6075](https://jira.mariadb.org/browse/MCOL-6075)] - Research/Reproduce 23.10.4 Memory Allocation Related Errors
* \[[MCOL-6291](https://jira.mariadb.org/browse/MCOL-6291)] - Cmapi Python now 3.13.11
