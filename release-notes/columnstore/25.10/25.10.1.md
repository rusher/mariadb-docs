# MariaDB ColumnStore 25.10.1 Release Notes

{% hint style="info" %}
ColumnStore 25.10.1 works with MariaDB Enterprise Server 11.8.
{% endhint %}

**Release date:** 6 Nov 2025

MariaDB ColumnStore 25.10.1 is a [Stable (GA)](../../community-server/about/release-criteria.md) release of MariaDB ColumnStore.

## Notable changes

* \[[MCOL-4605](https://jira.mariadb.org/browse/MCOL-4605)] - FROM\_UNIXTIME(negativeDecimal) not behaving as InnoDB
* \[[MCOL-4905](https://jira.mariadb.org/browse/MCOL-4905)] - make log message "Got a 4xx error" more verbose
* \[[MCOL-6194](https://jira.mariadb.org/browse/MCOL-6194)] - Cmapi user configurable failover no response wait time
* \[[MCOL-6181](https://jira.mariadb.org/browse/MCOL-6181)] - QA must handle table's primary keys statistics.
* \[[MCOL-6194](https://jira.mariadb.org/browse/MCOL-6194)] - Cmapi user configurable failover no response wait time
* \[[MCOL-5778](https://jira.mariadb.org/browse/MCOL-5778)] - SUM return NULL value due to NULLIF Function
