# MariaDB ColumnStore 23.10.7 Release Notes

## Overview

MariaDB Enterprise ColumnStore 23.10.7 is a maintenance release of MariaDB Enterprise ColumnStore. MariaDB Enterprise ColumnStore is a columnar storage engine included with MariaDB Enterprise Server.

MariaDB Enterprise ColumnStore 23.10.7 was released on 11 November 2025. This release is of General Availability (GA) maturity. MariaDB Enterprise ColumnStore 23.10.7 is a GA release in the 23.10 series.

This release of MariaDB Enterprise ColumnStore is included with MariaDB Enterprise Server 10.6.23-19 and MariaDB Enterprise Server 11.4.8-5.

## Notable changes

* \[[MCOL-4605](https://jira.mariadb.org/browse/MCOL-4605)] - FROM\_UNIXTIME(negativeDecimal) not behaving as InnoDB
* \[[MCOL-4905](https://jira.mariadb.org/browse/MCOL-4905)] - make log message "Got a 4xx error" more verbose
* \[[MCOL-6194](https://jira.mariadb.org/browse/MCOL-6194)] - Cmapi user configurable failover no response wait time
* \[[MCOL-5843](https://jira.mariadb.org/browse/MCOL-5843)] - initialize extent min/max with reasonable values
* \[[MCOL-5508](https://jira.mariadb.org/browse/MCOL-5508)] - make timeout configurable " A node is unresponsive for cmd = 4, no reconfigure in at least 300 seconds. Setting read-only mode."
* \[[MCOL-6105](https://jira.mariadb.org/browse/MCOL-6105)] - deactivate\_only logic is always enabled on node removal in CMAPI
* \[[MCOL-6100](https://jira.mariadb.org/browse/MCOL-6100)] - Extremely large primloc logs generated after upgrade from 23.10.3 to 23.10.4

## Issues fixed

### Can result in hang or crash

* \[[MCOL-6198](https://jira.mariadb.org/browse/MCOL-6198)] - Cross Engine PrimProc crash Signal 11
* \[[MCOL-6106](https://jira.mariadb.org/browse/MCOL-6106)] - Post upgrade to 23.10.4, some of the queries giving different set of results upon executing multiple times.
* \[[MCOL-5222](https://jira.mariadb.org/browse/MCOL-5222)] - ORDER BY on UNIONs in outer selects does not work
* \[[MCOL-6032](https://jira.mariadb.org/browse/MCOL-6032)] - Cmapi | check files paths on startup

### Related to performance

\[[MCOL-5756](https://jira.mariadb.org/browse/MCOL-5756)] - Handle UNION pushdown edge case that triggers slow table mode processing
