# MariaDB ColumnStore 23.10.8 Release Notes

## Overview

MariaDB Enterprise ColumnStore 23.10.8 is a maintenance release of MariaDB Enterprise ColumnStore. MariaDB Enterprise ColumnStore is a columnar storage engine included with MariaDB Enterprise Server.

MariaDB Enterprise ColumnStore 23.10.8 was released on 20 Jan 2026. This release is of General Availability (GA) maturity. MariaDB Enterprise ColumnStore 23.10.8 is a GA release in the 23.10 series.

This release of MariaDB Enterprise ColumnStore is included with MariaDB Enterprise Server 10.6.24-20.
