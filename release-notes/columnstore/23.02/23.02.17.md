# MariaDB ColumnStore 23.02.17 Release Notes

## Overview

MariaDB Enterprise ColumnStore 23.02.17 is a Stable (GA) maintenance release of MariaDB Enterprise ColumnStore. MariaDB Enterprise ColumnStore is a columnar storage engine that is included with the MariaDB Enterprise Server.

**Release date:** 19 Dec 2025
