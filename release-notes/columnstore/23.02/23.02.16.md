# MariaDB ColumnStore 23.02.16 Release Notes

## Overview

MariaDB Enterprise ColumnStore 23.02.16 is a Stable (GA) maintenance release of MariaDB Enterprise ColumnStore. MariaDB Enterprise ColumnStore is a columnar storage engine that is included with the MariaDB Enterprise Server.

**Release date:** 14 Oct 2025

## Notable changes

* \[[MCOL-6106](https://jira.mariadb.org/browse/MCOL-6106)] - Post upgrade to 23.10.4, some of the queries giving different set of results upon executing multiple times.
* \[[MCOL-5922](https://jira.mariadb.org/browse/MCOL-5922)] - NULLIF function returns -32768 instead of NULL when querying a Columnstore table
* \[[MCOL-5555](https://jira.mariadb.org/browse/MCOL-5555)] - Add support for readonly command which waits until all active cpimport jobs finished before putting system in readonly
* \[[MCOL-6104](https://jira.mariadb.org/browse/MCOL-6104)] - Backport dbrmctl startreadonly to 23.02.x

## Issues fixed

### Can result in hang or crash

* \[[MCOL-6126](https://jira.mariadb.org/browse/MCOL-6126)] - Prevent Data Loss: Local Extent Map Backup on NFS Outages
* \[[MCOL-6064](https://jira.mariadb.org/browse/MCOL-6064)] - Excessive and Unreleased memory
* \[[MCOL-6094](https://jira.mariadb.org/browse/MCOL-6094)] - controlernode left shmem locks behind forcing save\_brm to fail b/c of the locks
