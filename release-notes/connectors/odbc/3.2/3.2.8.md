# Connector/ODBC 3.2.8 Release Notes

<a href="https://mariadb.com/downloads/connectors/connectors-data-access/odbc-connector" class="button primary">Download</a> <a href="3.2.8.md" class="button secondary">Release Notes</a> <a href="../changelogs/3.2/3.2.8.md" class="button secondary">Changelog</a> <a href="https://app.gitbook.com/s/CjGYMsT2MVP4nd3IyW2L/mariadb-connector-odbc" class="button secondary">About MariaDB Connector/ODBC</a>

**Release date:**  2025

This is a [Stable (GA)](../../../community-server/about/release-criteria.md) release of MariaDB Connector/ODBC 3.2.

MariaDB Connector/ODBC 3.2.8 is built on top of [MariaDB Connector/C v.3.4.8](../../c/3.4/3.4.8.md).

## Bug Fixes 
* [ODBC-476](https://jira.mariadb.org/browse/ODBC-476) - SQLSetPos can delete unintended row(s) if the available index is unique and part of the key for the deleted record is NULL 
* [ODBC-479](https://jira.mariadb.org/browse/ODBC-479) - Crash at the attempt to read out of range column number
* [ODBC-480](https://jira.mariadb.org/browse/ODBC-480) - Incorrect work of SQLDescribeCol with NULL name buffer
* [ODBC-481](https://jira.mariadb.org/browse/ODBC-481) - ADODB crashes on the call to SQLCancel
* [ODBC-482](https://jira.mariadb.org/browse/ODBC-482) - Problem with SQLSetPos on block cursor with NULL values
