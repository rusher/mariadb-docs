## Notable changes

* [CONJ-1282](https://jira.mariadb.org/browse/CONJ-1282) Added TLS SNI (Server Name Indication) support
* [CONJ-1271](https://jira.mariadb.org/browse/CONJ-1271) Added MariaDbDataTruncation exception that includes MariaDB error code and error message

## Bugs Fixed

* [CONJ-1291](https://jira.mariadb.org/browse/CONJ-1291) Fixed ConnectionPoolDataSource incompatibility and connection leaks due to internal ConnectionEventListener in MariaDB Connector/J 3.5.x
* [CONJ-1286](https://jira.mariadb.org/browse/CONJ-1286) Fixed MariaDbPoolPinnedConnection thread-safety issue
* [CONJ-1283](https://jira.mariadb.org/browse/CONJ-1283) Fixed rewriteBatchedStatements introduced in 3.5.6 bug when having parenthesis after VALUES parts
* [CONJ-1287](https://jira.mariadb.org/browse/CONJ-1287) Fixed authentication plugin multi-exchange prefix (0x01) handling introduced in MDEV-37554
* [CONJ-1290](https://jira.mariadb.org/browse/CONJ-1290) Fixed NullPointerException in MultiPrimaryReplicaClient
* [CONJ-1295](https://jira.mariadb.org/browse/CONJ-1295) Fixed MariaDbClob compilation with JDK 26+
